스택
    삽입과 삭제 연산이 후입선출(LIFO)로 이루어지는 자료구조
    append -> Top위치에 데이터 삽입
    pop -> Top위치의 데이터 삭제 및 확인
    [-1] -> Top위치의 데이터를 단순 확인
    DFS(우선 탐색), 백트래킹 종류의 코테에 효과적이다.
    후입선출의 개념이 재귀 함수 알고리즘 원리와 일맥 상통한다.

큐
    삽입과 삭제 연산이 선입선출(FIFO)로 이루어지는 자료구조
    Python 에서는 deque를 이용하여 큐를 구현
    또는 list로도 구현 가능(deque가 시간복잡도에서 더 유리함)
    rear -> 큐에서 가장 끝 데이터
    front -> 큐에서 가장 앞의 데이터
    append -> rear에 데이터 삽입
    popleft -> front의 데이터를 삭제 및 확인
    [0] -> 큐의 front의 데이터를 확인
    큐는 너비 우선 탐색(BFS)에서 자주 사용함
우선 순위 큐
    들어간 순서 상관 없이 우선순위가 높은 데이터가 먼저 나오는 자료구조
    heap을 이용하여 구현 -> MAX, MIN으로 구현하는 차이가 있음
    front에 항상 최댓값 또는 최솟값이 위치함
